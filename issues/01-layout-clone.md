# Layout Clone (Oona.es)

**Status:** Open
**Personas:** @frontend-master, @ui-ux-designer

## Objective
Clone the baseline UI shell of `oona.es` for Cusco Bienestar. This task is purely frontend and establishes the visual language for the other tasks.

## Tasks
1. Set up the generic `app/layout.tsx` to include a clean, minimalist Header and Footer.
2. The Header should have a logo space on the left and a navigation menu on the right (Events, Professionals, About).
3. Update Tailwind typography and colors to be neutral and earthy (wellness theme) using predominantly browns and greens.
4. Create a responsive container wrapper component that other pages can use.

## Validation
- Ensure no layout shifts occur on mobile and desktop.
- Header must stick to the top if needed.
- `npm run build` and `npm run test` must pass.
